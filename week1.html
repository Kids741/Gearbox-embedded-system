Week 1 – Gearbox Embedded Systems Journey

Documenting my learning experience at Gearbox.

Introduction

The first week at Gearbox was focused on orientation, refreshing core development tools, and understanding the structure of the Embedded Systems program. We were introduced to the course roadmap and the philosophy behind the learning process.

Course Overview

During the introduction, we explored what embedded systems are and how they power many of the technologies used today, including medical devices, robotics, IoT systems, and smart infrastructure.

The instructors emphasized a hands-on approach to learning and encouraged us to think like builders and innovators rather than just students. One of the key ideas shared was that with the right mindset and tools, we can build almost anything.

Revision of Core Web Technologies

HTML

We revisited the fundamentals of HTML and how it forms the structure of web pages. This included basic elements such as headings, paragraphs, sections, lists, and semantic tags used to organize content clearly.

Git

Git was introduced as an essential version control system that helps developers track changes in code, collaborate effectively, and maintain a history of project development.

GitHub

We also explored GitHub as a platform for hosting repositories and collaborating on projects. We practiced pushing code, managing repositories, and using GitHub as a portfolio to showcase our work.

Key Insight from Week 1

One powerful idea emphasized during the week was inspired by the MIT approach to learning: the belief that with curiosity, persistence, and access to knowledge, it is possible to build almost anything.

This mindset encourages experimentation, creativity, and solving real-world problems through engineering and technology.

Reflection

Week 1 set a strong foundation for the journey ahead. It reminded me of the importance of mastering fundamental tools like Git and HTML while also developing a builder’s mindset.

As someone passionate about healthcare technology, I look forward to applying embedded systems knowledge to build innovative solutions that can improve patient care and medical technology.
